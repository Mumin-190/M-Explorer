### Project Summary (For GitHub "About" section)
M-Explorer is an advanced, offline-first Android file manager built with Kotlin and Material Design 3. It bridges the gap between everyday file management and power-user tooling by offering ultra-fast MediaStore indexing, hardware-encrypted SFTP networking, and secure, sandbox-compliant root filesystem manipulation via Shizuku. Completely free of telemetry, ads, and analytics.
